# sc-site
the main, lean, completely custom website for myself. 
framework-less, just libraries. 


## stack 
- ts
- vite
- tailwind css
- three.js
- theatre.js
- swiperjs
- baffle text (TBD)
- ...


## live URLs
- https://spencercap.github.io/sc-site/
- https://www.spencercappiello.com/ 


## DEV

- install:
```
pnpm i
```

- server:

```bash
pnpm run dev
```

- prod build:

```bash
pnpm run build
```
